# 公司新闻脉搏：股价异动快速归因

**定位**：快速情报响应，**非深度投研**。
**目标**：10-15分钟内回答：公司最近发生了什么？股价异动的真因是什么？要不要重审投资论文？

## 适用场景

- ✅ 股价单日异动±5%或一周±10%
- ✅ 财报后异动
- ✅ 需辨别新闻噪音与信号
- ❌ 完整投研（用 investment-team）
- ❌ 财报深读（用 earnings-review）

## 执行流程

### 第一步：确认参数

从用户消息提取：
- **公司名**（必填，如未提供用 clarify 询问）
- **时间窗口**（默认14天，财报季可缩至7天）
- **股价异动详情**（选填，用于聚焦归因）

### 第二步：信息可得性分级

| 等级 | 特征 | 侦察策略 |
|------|------|---------|
| A级 | 大盘股，信息充裕 | 降噪和归因，过滤重复二手新闻 |
| B级 | 中小盘，信息适中 | 标准模式，关键事件附独立信源 |
| C级 | 冷门股，信息稀缺 | "扫盲模式"，"找不到新闻"本身即有价值 |

### 第三步：并行侦察（delegate_task）

使用 `delegate_task` 并行启动 3 个侦察 Agent：

```
delegate_task(tasks=[
  {
    "goal": "侦察 {公司名} 近{时间窗口}天的公司本体事件",
    "context": "你负责侦察公司本体事件维度。搜索：官方公告、财报、管理层动作、重大业务事件、资本运作、诉讼合规。输出事件时间线表格，每条标注 日期/来源/摘要/与异动相关性。优先用 web_search 搜索时效性信息。传言需至少2个独立信源验证。严格区分事实与推测。",
    "toolsets": ["web"]
  },
  {
    "goal": "侦察 {公司名} 近{时间窗口}天的监管政策与行业对手动态",
    "context": "你负责监管政策+行业对手两个维度。搜索：行业新规、跨境政策、税收、反垄断、直接对手事件、产业链上下游变化、行业景气度。判断：是个体事件还是行业Beta波动？政策直接影响公司程度（直接/间接/无关）。优先用 web_search 搜索时效性信息。",
    "toolsets": ["web"]
  },
  {
    "goal": "侦察 {公司名} 近{时间窗口}天的市场情绪与机构观点",
    "context": "你负责市场情绪维度。搜索：卖方评级变动、机构持仓变化、做空数据、大V观点、传言小作文、技术面信号。关键判断：是基本面驱动还是情绪/资金面驱动？优先用 web_search 搜索时效性信息。",
    "toolsets": ["web"]
  }
])
```

### 第四步：综合归因报告

汇总三个维度报告后，输出结构化报告：

#### 1. 一句话归因
30-60字说明异动主因、次因及性质（价值事件/情绪波动/不明）。

#### 2. 合并事件时间线

| 日期 | 维度 | 事件 | 来源 | 异动归因权重 |
|------|------|------|------|------------|
| | | | | 🔴高 / 🟡中 / ⚪低 |

#### 3. 异动归因表

| 候选解释 | 证据 | 反证 | 置信度 | 持续性 |
|----------|------|------|--------|--------|

#### 4. 性质判断（核心结论）
- [ ] **价值事件**：基本面真实变化，需重审论文
- [ ] **情绪/技术波动**：基本面无变化，属资金面/情绪驱动
- [ ] **真因不明**：最危险结论，可能市场提前知道信息
- [ ] **混合**：部分价值事件+情绪放大

#### 5. 行动建议清单

| 行动 | 是否建议 | 理由 |
|------|----------|------|

#### 6. 接下来7-30天跟踪清单

### 第五步：保存报告

写入 `~/workspace/ai-berkshire/reports/{公司名}/{公司名}-news-{YYYYMMDD}.md`

## 关键原则

1. **快胜过全**：10-15分钟内给出归因判断是核心价值
2. **归因优先于罗列**：判断"哪个事件配得上这次异动"，要做减法
3. **诚实面对"不明"**：找不到主因时明确写"真因不明"
4. **不预设立场**：证据指向哪边就写哪边
5. **区分"催化剂"和"巧合"**：事件需与异动影响量级匹配
