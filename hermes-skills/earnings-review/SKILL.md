# 财报精读：像巴菲特一样读年报

对用户指定公司的最新财报进行深度精读分析。

如用户未提供公司名和财报期间，用 `clarify` 询问。

## 执行流程

### 第一步：获取财报原文

使用 `web_search` 搜索该公司的最新季度/年度财报。优先获取：
- 公司官网投资者关系页面的财报原文
- SEC/港交所/巨潮资讯的原始文件

使用 `web_extract` 提取财报内容。

### 第二步：数据提取与验证

提取关键财务数据，并通过工具验证：

```bash
python3 ~/workspace/ai-berkshire/tools/financial_rigor.py verify-valuation \
  --price {股价} --eps {EPS} --bvps {每股净资产}

python3 ~/workspace/ai-berkshire/tools/financial_rigor.py cross-validate \
  --field revenue --values '{\"年报\": 7518, \"Yahoo\": 7500}' --unit 亿
```

### 第三步：逐模块分析

#### 3.1 收入分析
- 各业务线收入拆解
- 同比/环比变化
- 收入结构变化趋势

#### 3.2 利润分析
- 毛利率、经营利润率、净利率趋势
- 费用结构变化（研发/销售/管理）
- 非经常性损益影响

#### 3.3 现金流分析
- 经营性现金流 vs 净利润
- 自由现金流
- 资本开支变化

#### 3.4 资产负债表
- 现金储备变化
- 负债结构
- 应收账款/存货变化

#### 3.5 管理层展望
- 电话会议关键信息
- 未来指引
- 战略方向变化

### 第四步：关键判断

- 本季度是超预期还是低于预期？
- 最大的亮点和最大的隐患是什么？
- 对投资论文的影响（强化/弱化/不变）

### 第五步：保存报告

写入 `~/workspace/ai-berkshire/reports/{公司名}/{公司名}-earnings-{期间}.md`

## 输出格式

1. **一句话总结**：50字内概括本季财报核心信息
2. **关键数据表**：本季 vs 上季 vs 去年同期
3. **分模块详细分析**
4. **超/低于预期的具体项目**
5. **对投资论文的影响**
6. **下一步关注点**
